## 一、品牌定位

### Logo 设计理念

#### 设计风格
参考智谱 AI 的 Z Logo 设计风格，采用几何斜切矩形组合。

#### Logo 结构
```
┌─┐
│ └──────┐
│       /
│     /      ← 上斜线（斜切矩形）
│   /
│ /
│ \
│   \
│     \      ← 下斜线（斜切矩形）
│       \
│ ┌──────┘
└─┘
↑
竖线
```

#### 组成元素
| 元素 | 形状 | 含义 |
|------|------|------|
| 竖线 | 矩形 | 稳固的基础 |
| 上斜线 | 斜切矩形 | 向上的力量 |
| 下斜线 | 斜切矩形 | 向下的延伸 |
| 中间留白 | 断开空间 | 象征"连接"——等待被连接的空间 |

#### 设计理念
1. **K 字识别性**：清晰可辨的 K 字母形状
2. **断开留白**：中间的断开空间象征"连接"的概念，呼应品牌 Slogan "连接世界"
3. **几何现代感**：斜切矩形组合，简洁大气
4. **与"启霖智联"融合**：K 图标与中文字体大小协调，上下对齐

#### 字体搭配
- K 图标高度：28px
- 中文字体：26px，PingFang SC / Microsoft YaHei
- 字重：600（Semi-bold）
- 字间距：2px

|------|------|
| 白色背景导航栏 | 深灰 `#1E293B` |
| 深色背景 | 白色 `#FFFFFF` |
